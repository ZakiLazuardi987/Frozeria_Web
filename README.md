# Frozeria_Web
Sistem ini dapat menangani data stok makanan dan kategorinya. Setiap kali ada  barang masuk, staf akan menginputkan barang tersebut kedalam sistem. Jika barang  tersebut adalah barang baru, maka staf harus menginputkannya ke data barang. Jika  barang tersebut sudah tercatat, maka staf akan mengubah jumlah stok nya sesuai  dengan kondisi saat ini.
